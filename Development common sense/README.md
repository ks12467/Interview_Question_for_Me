# 개발 기초 상식

---
## 목차


---
### 컴파일과 인터프리터의 차이는 무엇인가요?
컴파일은 전체 코드를 한 번에 기계어로 변환해 실행 파일을 만들고, 인터프리터는 한 줄씩 읽으며 바로 실행됩니다. <br>

---
### CPU와 메모리 간 병목을 줄이는 기술은 무엇인가요?
CPU와 메모리 간 속도 차이를 줄이기 위해 캐시 메모리를 사용합니다. <br>
캐시는 자주 사용하는 데이터를 저장해 CPU가 빠르게 접근할 수 있도록 도와줍니다. <br>
또한, 파이프라이닝과 멀티코어 구조도 병목 현상을 완화하는 데 기여합니다. <br>

---
### 동기와 비동기의 차이는 무엇인가요?
동기는 한 작업이 끝나야 다음 작업이 실행되고, 비동기는 기다리지 않고 다음 작업을 병렬로 진행됩니다. <br>

---
### REST API란 무엇인가요?
REST API는 HTTP 프로토콜을 기반으로 클라이언트와 서버 간에 데이터를 주고받는 아키텍처 스타일입니다. <br>
자원은 URI로 식별되고, HTTP 메서드(GET, POST, PUT, DELETE 등)를 사용해 자원을 조작합니다. <br>
REST는 무상태성(stateless)을 유지하며, 클라이언트와 서버 간의 독립성을 보장합니다. <br>
