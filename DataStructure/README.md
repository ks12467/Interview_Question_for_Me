## 자료구조란 무엇인가요?
자료구조는 데이터를 효율적으로 저장하고, 접근﹒변경﹒탐색하기 위한 구조적 방식입니다 <br>
즉 데이터를 어떻게 담거, 꺼낼지를 정의하는 체계

심화 : <br>
좋은 자료구조는 시간 복잡도와 공간 효율성의 균형을 맞추는 것이 핵심입니다.
예를 들어, 탐색 속도가 중요한 경우 -> 해시, 삽입﹒삭제가 많은 경우 -> 링크드 리스트를 사용

---
## 배열과 연결 리스트의 차이는 무엇인가요?
배열은 연속된 메모리 공간에 데이터를 저장하여 인덱스로 직접 접근할 수 있습니다. <br>
반면 연결 리스트는 노드들이 포인터로 연결된 구조로, 메모리가 비연속적입니다.<br>
